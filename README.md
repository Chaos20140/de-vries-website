# de Vries — Website

Neugestaltete Website für **de Vries** (Salzhemmendorf) – Familienbetrieb seit 1998 für
**Garten- & Landschaftsbau**, **Seniorenbetreuung** und **Haushaltshilfe**.

Statische Multipage-Site (HTML / CSS / Vanilla-JS), kein Build-Step. „Editorial Prestige"-Design
mit Markenrot `#d7120a`, Fraunces + Manrope, smoothem Scrollen (Lenis) und Scroll-Animationen.

## Lokal ansehen
```bash
python -m http.server 8848
# → http://127.0.0.1:8848/index.html
```

## Seiten
`index.html` · `garten-landschaftsbau.html` · `seniorenbetreuung.html` · `entlastungsbetrag.html` ·
`pflegeleistungen.html` · `haushaltshilfe.html` · `referenzen.html` · `kontakt.html` ·
`stellenangebote.html` · `impressum.html` · `datenschutz.html`

## Struktur
- `assets/css/style.css` — komplettes Stylesheet (Single-File)
- `assets/js/main.js` — Animations-Engine (Reveals, Counter, Magnetic, Timeline, Mobile-Nav, Formular)
- `assets/js/lenis.min.js` — Smooth-Scroll (lokal)
- `assets/img/` — Logo, Favicon, Original-Fotos
- `CLAUDE.md` — interne Bauanleitung
